## Objective
To understand how to consume public REST APIs using Python and handle JSON data.

## Tools Used
- Python
- requests library
- VS Code

## API Used
Public Quotes API: https://api.quotable.io/random

## Features
- Sends GET request to public API
- Checks HTTP status codes
- Parses JSON response
- Extracts required data
- Handles API errors gracefully
- Saves API response to local JSON file
- Displays clean formatted output

## How to Run
1. Install dependencies:
   pip install requests
2. Run the script:
   python api_data_fetch.py

## Output
- Displays a random quote and author
- Saves full JSON response to api_response.json

## Learning Outcome
Understanding REST API communication, HTTP methods, status codes, and JSON parsing in Python.
